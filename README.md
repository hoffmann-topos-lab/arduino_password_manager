# arduino_password_manager

#English

#Arduino Password Manager

#Description

This Arduino project is a password manager that allows users to store, view, and delete passwords for various services such as Email, Instagram, Facebook, among others. It utilizes an LCD display for user interface and buttons for navigation.

#How to Use

#Installation
Connect the LCD display and buttons to your Arduino Uno as specified in the code.
Upload the gerenciador_v4.ino sketch to your Arduino.

#Usage
Use the btnUp and btnDown buttons to navigate through menu options.
Press btnSelect to select an option.
In the main menu, you can add new passwords, view stored passwords, or delete an existing password.

#Português

#Gerenciador de Senhas Arduino

#Descrição

Este projeto Arduino é um gerenciador de senhas que permite aos usuários armazenar, visualizar e deletar senhas de diferentes serviços, como Email, Instagram, Facebook, entre outros. Utiliza um display LCD para a interface do usuário e botões para navegação.

##Como Usar

#Instalação
Conecte o display LCD e os botões ao seu Arduino Uno conforme especificado no código.
Faça o upload do sketch gerenciador_v4.ino para o seu Arduino.

#Uso

Use os botões btnUp e btnDown para navegar pelas opções do menu.
Pressione btnSelect para selecionar uma opção.
No menu principal, você pode adicionar novas senhas, visualizar as senhas armazenadas ou deletar uma senha existente.
